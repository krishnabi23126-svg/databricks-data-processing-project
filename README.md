# Databricks Data Processing Project

## Overview
This project demonstrates real-time data processing using Azure Databricks and PySpark.

## Tech Stack
- Azure Databricks
- PySpark
- JSON Data

## Workflow
1. Ingest streaming JSON data
2. Process data using PySpark
3. Aggregate sales data
4. Output results in real-time

## Key Features
- Streaming data processing
- Aggregation logic
- Scalable architecture

## Use Case
- Real-time sales analytics
- Product performance tracking

## Future Enhancements
- Integrate with Kafka
- Store output in Delta Lake
- Connect with Power BI
